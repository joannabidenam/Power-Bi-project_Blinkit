# Tasks Accomplished

## Data Preparation & Modeling

* Cleaned and transformed raw data using Power Query
* Handled missing values using techniques such as median replacement and logic-based DAX (IF, SWITCH)
* Structured the dataset by refining rows/columns and removing irrelevant fields
* Established relationships to create a reliable and dynamic data model

### Calculations & KPIs

* Created explicit DAX measures for better control and reusability
* Built key performance metrics including:

  * **Total Sales**
  * **Average Sales**
  * **Total Items Sold**

### Visualisation & Dashboard

* Designed interactive dashboards with insightful charts and slicers
* Created parameters for dynamic filtering and enhanced user experience
* Analyzed:

  * Sales performance trends
  * Outlet performance by location, type, and size
  * Customer demand patterns and product movement
